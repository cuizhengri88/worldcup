# 2026 FIFA 월드컵 토너먼트 예측 시뮬레이터 분석

## 개요
`2026.html`은 2026 FIFA 월드컵 토너먼트 예측을 시각화하는 단일 페이지 HTML 문서입니다. Bootstrap 5와 FontAwesome을 사용하며, 어두운 테마의 토너먼트 브래킷 시각화와 모바일 대응형 레이아웃을 제공합니다.

## 주요 기능
- 토너먼트 대진표 표시
- 각 경기마다 승리 확률(%)과 승패 상태 표시
- 팀 클릭 시 분석 이유를 모달로 표시하는 인터랙션
- PC에서는 가로 스크롤 토너먼트, 모바일에서는 세로 레이아웃으로 전환
- 대한민국 경기와 결승전 등 특정 매치를 강조하는 스타일

## 파일 구조
1. `<head>`
   - 메타 정보: `charset`, `viewport`, `title`
   - 외부 CSS: Bootstrap 5, FontAwesome
   - 내부 스타일: 토너먼트, 팀 카드, 모바일 대응 미디어 쿼리 등
2. `<body>`
   - 히어로 섹션: 제목과 안내 문구
   - 대진표 컨테이너: `.bracket-scroll-container` 안에 `.tournament-bracket`
   - 8개 라운드 열(`.bracket-column`) 구성
   - 분석 모달: `#analysisModal`
3. `<script>`
   - `showReason(teamName, odds, reasonText)` 함수
   - 모달 텍스트 삽입 후 Bootstrap Modal 표시

## 스타일 요약
- 컬러 팔레트: 어두운 배경(`--bg-main`, `--bg-card`), 파란색/주황색 강조, 한국팀 핑크 강조
- 카드 스타일: 둥근 모서리, 그림자, 호버 변환
- 승자/패자 클래스: `.winner` / `.loser`로 상태 색상과 텍스트 스타일 변경
- 특별 강조:
  - `.korea-highlight`: 대한민국 경기 강조
  - `.final-stage`: 결승전 강조
- 반응형
  - `max-width: 767.98px`: 브래킷을 세로 스택으로 변경, 모바일에 적합한 라운드 헤더 스타일
  - `min-width: 768px`: 가로 스크롤과 큰 최소 너비 유지

## 토너먼트 라운드 구성
- 32강 좌측 블록 (A조~D조): MATCH 01~04
  - 멕시코, 일본, 포르투갈, 잉글랜드가 예측 승리
- 16강 좌측 블록: 일본, 잉글랜드 예측 승리
- 8강 좌측 쿼터: 잉글랜드 예측 승리
- 준결승: 잉글랜드 vs 브라질, 프랑스 vs 아르헨티나
  - 잉글랜드, 프랑스 예측 승리
- 결승: 잉글랜드 vs 프랑스
  - `FRANCE WIN (2-1)` 예측
- 8강 우측 블록: 프랑스 승리
- 16강 우측 블록: 대한민국 vs 프랑스, 독일 vs 네덜란드
  - 프랑스, 독일 예측 승리
- 32강 우측 블록: 대한민국, 프랑스, 독일, 네덜란드 승리

## 주요 인터랙션
- 각 `.team-row-btn` 클릭 시 `showReason()` 호출
- 모달 제목에 팀 이름 적용
- 모달에 승률과 분석 이유 표시
- Bootstrap 5 모달을 사용하여 표시

## 주목할 부분
- `showReason()` 함수는 단일 HTML `<script>`에 정의되어 있으며, 모달 표시 로직이 간단합니다.
- HTML은 정적 구조로 직접 경기 데이터를 하드코딩하고 있습니다.
- 모바일 대응을 위해 커스텀 미디어 쿼리가 상세하게 적용되어 있습니다.

## 결론
이 페이지는 2026 월드컵 경기 예측 시뮬레이터를 시각적으로 구성하며, 클릭 가능한 팀 카드로 각 팀의 예측 이유를 보여주는 UI 중심의 데모입니다. 추가 개선 시에는 데이터 구조(예: JSON)로 경기 데이터를 분리하고, JavaScript로 동적 렌더링을 적용하면 확장성이 더 좋아집니다.
